## Personal projects

I like to do a lot of random/fun coding projects

- [github burndown](https://colindiesh.shinyapps.io/github_burndown/) a shiny app to plot "burndown" style statistics on github issues (2021)
- [myloveydove.com](https://myloveydove.com) a photo gallery for dixie (RIP). has video/photo uploads/commenting with AWS lambda and S3 (2020, [source](https://github.com/cmdcolin/aws_serverless_photo_gallery/))
- [logistic_chaos_map](https://cmdcolin.github.io/logistic_chaos_map/) logistic map fractal viewer with WASM and JS drawing (2020)
- [graphgenomeviewer](https://cmdcolin.github.io/graphgenomeviewer) BCC2020 hackathon project to visualize GFA graph genome files (2020)
- [githubgraphjs](https://cmdcolin.github.io/githubgraphjs) visualize your Github Actions build times (2020)
- [travigraphjs](https://cmdcolin.github.io/travigraphjs) visualize your travis-CI build times (2019)
- [gratiotquest](https://github.com/vastholdings/gratiotquest) multi-player websocket RPG where you are a pheasant running around a MS Paint world (2018)
- [slap happy](https://github.com/cmdcolin/slaphappy) retrofitted a arcade cabinet with a raspberry pi (2018)
- [resort](https://cmdcolin.github.io/resort/qs.html) visualize sorting algorithms (2017)
